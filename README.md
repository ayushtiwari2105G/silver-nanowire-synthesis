# 🧪 Large-Scale Synthesis of Silver Nanowires using Continuous Flow Reactor

## 📌 Project Overview

This project focuses on the **process-intensified synthesis of Silver Nanowires (AgNWs)** using a **continuous flow reactor system**. The aim is to develop a **scalable, cost-effective, and efficient method** for producing nanomaterials used in advanced electronic applications.

This work is inspired by recent research in nanotechnology and chemical engineering, emphasizing **high-throughput manufacturing and process optimization**.

---

## 🎯 Objectives

* To synthesize **high-quality Silver Nanowires (AgNWs)**
* To study the effect of:

  * Temperature
  * Residence time
  * Silver-to-chloride ion ratio
  * Capping agent (PVP) concentration
* To develop **conductive ink formulations**
* To explore **scalable industrial production methods**

---

## 🔬 Materials Used

* Silver Nitrate (AgNO₃)
* Ethylene Glycol (EG)
* Cupric Chloride (CuCl₂)
* Polyvinylpyrrolidone (PVP)
* Methanol, Acetone

---

## ⚙️ Methodology

### 1. Preparation of Precursor Solution

* PVP dissolved in Ethylene Glycol
* Addition of AgNO₃ and CuCl₂
* Formation of precursor solution (AgPVPCl_EG)

### 2. Continuous Flow Synthesis

* Solution pumped through PTFE tube reactor
* Temperature maintained at **~150°C**
* Flow rate: **2 mL/min**
* Residence time: **~20 minutes**

### 3. Product Recovery

* Cooling to room temperature
* Centrifugation (5000 rpm)
* Washing with methanol
* Drying at 70°C

📌 The process converts silver ions → silver nanoparticles → nanowires

## 📊 Key Results

### ✅ Effect of Parameters

| Parameter      | Observation                          |
| -------------- | ------------------------------------ |
| Temperature    | Optimal at ~150°C                    |
| Residence Time | 20 min gives best nanowire formation |
| Ag:Cl Ratio    | Controls morphology and length       |
| PVP MW         | Higher MW reduces diameter           |

* Nanowire length increased up to **~50 µm**
* Diameter reduced to **~165 nm**
* Yield achieved up to **~90%**
